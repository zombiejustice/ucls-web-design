How to connect to our server from a Windows computer

  1. Download WinSCP from http://winscp.net/download/winscp433setup.exe
  2. Install WinSCP. If you have the option, choose 'explorer view' rather than 'commander view'.
  3. Make a new SFTP connection (not FTP - it won't work!). The 'host name' is 128.135.91.101
  4. Use the same name and password as you do in the classroom.
  5. Navigate in WinSCP to your Sites folder.
  6. You can right-click on your files to edit them right in WinSCP, or you can drag-and-drop new files into the window.

If any of these instructions are confusing or don't work, ask Mr. H about them tomorrow.
